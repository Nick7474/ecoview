🏠 사이트 소개

광명시 강소형 스마트도시
통합 플랫폼 에코뷰

에코뷰(EcoView)는 광명시의 탄소중립과 스마트도시 비전을 실현하기 위해 구축된 통합 개방형 플랫폼입니다. 시민·기업·연구자가 함께 도시 문제를 해결하고 데이터를 공유하며, 더 나은 광명시의 미래를 만들어 갑니다.

⭐ FEATURES

주요 기능 및 서비스

🌡

실시간 탄소·환경 현황

광명시 전체 탄소 배출·흡수, 에너지, 미세먼지 등 환경 데이터를 실시간으로 모니터링합니다.

📂

공공 데이터 허브

488건의 광명시 스마트도시 데이터를 개방하여 누구나 검색·다운로드·API 형태로 활용할 수 있습니다.

🧪

시민 참여 플랫폼

리빙랩·오픈랩·시티즌교육을 통해 시민이 직접 도시 문제 해결에 참여할 수 있습니다.

📊

도시지표 대시보드

탄소지표·스마트도시 지표를 시각화하여 광명시의 발전 현황을 한눈에 파악합니다.

🗺

스마트 시티맵

CCTV·안전시설·편의시설 위치를 지도 기반으로 실시간 확인할 수 있습니다.

🔒

마이페이지

내 사업 신청 현황, 데이터 활용 내역, 관심 사업을 개인화된 대시보드로 관리합니다.
